# Incident Workflow

1. Stabilize the system.
2. Define impact and severity.
3. Capture timeline.
4. Identify recent changes.
5. Mitigate or roll back safely.
6. Verify recovery.
7. Run root cause analysis.
8. Create follow-up actions.

## Output

- Current status.
- Impact.
- Root cause.
- Actions taken.
- Preventive work.
