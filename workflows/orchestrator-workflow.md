# Orchestrator Workflow

1. Intake objective and definition of done.
2. Identify risks and required approvals.
3. Choose specialist agents.
4. Create stage plan.
5. Execute discovery before implementation.
6. Validate stage outputs.
7. Integrate final answer.
8. Record durable decisions in memory when safe.

## Agent Pattern

```text
Researcher -> Software Architect -> Implementer -> Code Reviewer -> Documentation Writer
```

Use fewer agents when the task is simple.
