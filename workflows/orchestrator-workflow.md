# Orchestrator Workflow

Use when a personal Codex task has multiple stages or crosses coding, research, business, productivity, finance, deployment, or documentation.

1. Understand the request.
2. Choose the right agent or skill.
3. Create a short plan.
4. Execute safely.
5. Validate the work.
6. Give clear next steps.

## Common Agent Routes

```text
App build: Orchestrator -> Software Architect -> Full Stack Developer -> Code Reviewer
AI app: Orchestrator -> AI App Builder -> Full Stack Developer -> Code Reviewer
Research: Orchestrator -> Researcher -> Technical Documentation Writer
Automation: Orchestrator -> Automation Engineer -> GitHub Automation Specialist or Full Stack Developer
Career: Orchestrator -> Researcher -> Resume Career Coach
Finance: Orchestrator -> Finance Planning Advisor -> Technical Documentation Writer
Event/lifestyle: Orchestrator -> Researcher -> Event Production Advisor or Lifestyle Entertainment Planner
```

Use fewer agents when the task is simple.
