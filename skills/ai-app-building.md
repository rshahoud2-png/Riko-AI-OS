# AI App Building Skill

## Purpose
Design and build useful AI app features without overengineering.

## Use When
Building chat apps, copilots, AI workflows, prompt tools, RAG/search features, model-backed automations, or agentic prototypes.

## Inputs
- User problem and target workflow.
- Data sources and privacy limits.
- Model/provider assumptions.
- Expected outputs and failure cases.
- Deployment target.

## Method
1. Define the user job and success metric.
2. Choose the simplest pattern: prompt, structured output, tool call, RAG, workflow, or agent.
3. Design data flow and privacy boundaries.
4. Add guardrails, logging, and evaluation examples.
5. Ship an MVP before adding orchestration.

## Output
- AI feature plan.
- Data/model/tool flow.
- Implementation checklist.
- Evaluation examples.
- Risks and next steps.
